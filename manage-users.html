러닝스튜디오 기능별 가이드 Learning studio Manual

Step 1. 리소스 공유 관리 리스트

1. 러닝스튜디오 대시보드 접속합니다.

2. 좌측에 메뉴에서 관리 → 리소스 공유 관리 탭을 선택입니다.

Step 2. 리소스 공유 관리 상세 정보

    항목 설명
    계정 정보 로그인 계정 정보
    네임스페이스 현황 및 권한 네임스페이스에 대한 현황 리스트
    네임스페이스 권한 부여(리소스 권한) 현 네임스페이스 리소스 권한 부여
    권한 부여 : 이메일 형식으로 부여 가능

Step 3. 리소스 공유 관리 상세 정보

1) 계정 권한 부여

  • 권한 부여할 사용자 이메일 주소 등록

2) 계정 권한 부여 받은 사용자 권한 확인

  • 상단에 네임스페이스 리스트 확인

  • 자신의 네임스페이스 및 권한 부여 받은 네임스페이스 확인 가능

3) 권한 정상 부여 확인

  • 자신의 네임스페이스 리소스 확인



  • 부여 받은 네임스페이스 리소스 확인

Step 4. 리소스 공유 관리 권한 제거

  • 권한 제거할 이메일 주소 "x" 버튼 클릭
